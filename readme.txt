HOW TO USE:
1. Buka command prompt di file lokasi Battleships.exe
2. Jalankan command $ battleships -b "<This_Bot_Directory>" "<Other_Bot_Directory>"
3. Tambahkan atribut --clog --pretty untuk melihat jelasnya permainan
4. Enjoy!!
